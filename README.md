# Hangman

## Description
  Hangman game created as part of The Odin Project


## Installation
  * fork the repo
  * $ cd ~/hangman/
  * $ ruby lib/hangman.rb

## Usage
  * The game will then pop up.
  * Must have Ruby installed.
  * Game created and tested with Ruby 2.4.1.
  * Game also intended to be run from a unix command line.
